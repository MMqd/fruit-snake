A simple Linux terminal game where a snake eats fruit.

## Controls
h/j/k/l, Arrow Left/Down/Up/Right, n/e/i/o - Turn left/down/up/right
p,ESC - Pause/Unpause
SPACE - Unpause/Restart game after game over
